`notebooks/`, using their **saved outputs**. They are not executed at build time
because they need large datasets and heavy optional dependencies — open them in
Colab or Docker (see the [Colab guide](../colab_guide.md) and
[Docker guide](../docker_guide.md)) to run them yourself.
